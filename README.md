"Supervisión inteligente y programación visual para la robótica del futuro."

¿Qué es?

Un centro de mando web diseñado para monitorizar, programar y validar robots industriales de forma remota y sencilla.

¿Qué hace?

- Centraliza: Une diferentes centros (Salesianos, Urnieta) en una única interfaz visual.

 - Simplifica (Low-Code): Permite programar robots arrastrando bloques (Blockly) en lugar de escribir código complejo.

- Valida en Tiempo Real: Compara automáticamente la trayectoria teórica (el diseño) con la trayectoria real (la telemetría del robot) para detectar fallos o desviaciones al instante.

El objetivo?

Democratizar la robótica industrial, haciendo que programar y verificar un robot sea un proceso intuitivo, visual y libre de errores.
